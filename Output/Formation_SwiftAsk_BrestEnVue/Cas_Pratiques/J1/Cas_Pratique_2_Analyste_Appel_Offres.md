# Cas Pratique 2 : L'Analyste d'Appel d'Offres (J1 - Niveau N2)

*   **Objectif :** Créer un agent d'analyse documentaire avec instructions.
*   **Durée :** 45 min.
*   **Énoncé :** Vous devez analyser 3 réponses à un marché public pour une prestation de sécurité événementielle (fictif). Créez un agent qui lit le PDF de réponse et remplit une grille d'analyse prédéfinie.
*   **Input :** 1 fichier PDF (Offre prestataire sécurité) + Une grille de critères (Prix, Effectifs proposés, Références événements similaires).
*   **Critères de réussite :** L'agent sort un tableau comparatif ou une note de synthèse respectant strictement les critères demandés.
*   **Variante Facile :** Juste extraire le prix et le délai.
